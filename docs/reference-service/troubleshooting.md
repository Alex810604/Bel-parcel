# Устранение неполадок

## Ошибки 500 при вызове эндпоинтов
Причина: operator-api не может подключиться к reference-service
Решение:
1. Проверить доступность: curl http://reference-service:8084/healthz
2. Проверить логи: docker logs reference-service
3. Проверить конфигурацию Services.ReferenceURL в operator-api

## События не публикуются в Кафка
Причина: Сбой сети или недоступность брокера Kafka API (Redpanda)
Решение:
1. Проверить записи в outbox_events со статусом error
2. Проверить логи сервиса на наличие ошибок публикации
3. После восстановления Кафка события будут опубликованы автоматически (фоновая горутина)

## Нет аудита изменений
Причина: Не передан параметр reason в теле запроса
Решение: Все обновления должны содержать обязательное поле reason
